An error has occured :-(

Usual error causes

  1. You started the application from an IDE and you didn't run npm start or npm run webpack:build.
  2. You had a network error while running npm install. If you are behind a corporate proxy, it is likely that this error was caused by your proxy.
  3. You installed a Node.js version that doesn't work with spoon-ui: please use an LTS (long-term support) version, as it's the only version we support.
